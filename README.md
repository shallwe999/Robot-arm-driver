#### Serial Input

```C++
get          // get the pitch and roll angle
setpitch 90  // set pitch angle to 90 degree (45~235)
setroll 180  // set roll angle to 180 degree (90~270)
up           // raise the arm
down         // drop down the arm
qdown        // drop down the arm quickly

```

Baudrate: 9600
